# Build APK
Les exemples permettent de générer le .apk en local.  
**NON Fonctionnel** sur windows.
## Blank project
```sh
npm install -g eas-cli
eas login
eas build:configure
eas build -p android --local
```

## This project
```sh
npm install
eas login
eas build -p android --local
```
